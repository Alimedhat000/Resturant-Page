# Dynamic Restaurant Homepage

## Project Overview

This is a dynamic single-page restaurant website created using JavaScript and Webpack, where the entire page content is generated dynamically through DOM manipulation.
[Visit Here.](https://alimedhat000.github.io/Resturant-Page/)

## Features

- Fully dynamic content generation using JavaScript
- Tabbed navigation between Home, Menu, and Contact pages
- Webpack-based build process
- No hard-coded HTML content after initial setup

## Setup and Installation

### Prerequisites

- Node.js
- npm (Node Package Manager)

### Installation Steps

1. Clone the repository
   ```bash
   git clone <your-repository-url>
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start development server
   ```bash
   npx webpack serve
   ```

4. Build for production
   ```bash
   npx webpack
   ```
